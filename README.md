# UPI Spending Tracker

Analyzed 3 months of personal UPI transaction data using Python.

## Tools Used
- Python, Pandas, Matplotlib, Seaborn, pdfplumber

## What I Found
- Total Spent: ₹53,727 across 86 transactions
- Top category: Education (52% of spending)
- Analyzed spending trends across March-May 2026

## Files
- analysis.py - extracts and parses PDF
- graphs.py - generates 5 visualizations
